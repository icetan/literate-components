# Literate Components

Generatet interactive documentation for your web UI components using Markdown.

## Installation

```sh
git clone https://github.com/icetan/literate-components.git
cd literate-components
npm i -g
```

## Usage

```sh
lc --help
```

## Examples

To build the examples:

```sh
cd examples/
npm i
npm run docs
ls www/
```
